# Chess with Stockfish - Educational Analysis Tool

A Python-based chess analysis tool with GUI that integrates Stockfish engine for learning and analysis purposes.

## 📋 Overview

This project provides an educational chess analysis platform with a console-based GUI. It allows users to:
- Analyze chess positions using the Stockfish engine
- Study game positions offline
- Learn chess strategies through engine evaluation
- Understand engine recommendations and move analysis

**Note:** This tool is designed for **offline analysis and educational purposes only**. It should not be used for online play on any chess platform, as using engines during online games violates the terms of service of chess platforms like Chess.com.

## ⚠️ Important Disclaimer

- **Educational Use Only**: This project is intended for learning chess analysis and understanding how chess engines work
- **Not for Online Play**: Do not use this tool to gain unfair advantages in online chess games
- **Terms of Service**: Using this tool for online chess platforms violates their terms of service and can result in account bans
- **Fair Play**: Always play fairly and respect the rules of chess communities you participate in

## ✨ Features

- **Stockfish Integration**: Uses the powerful Stockfish chess engine for analysis
- **Console GUI**: User-friendly interface for analysis
- **Position Evaluation**: Get engine evaluation of any chess position
- **Move Analysis**: Understand the best moves and why
- **Educational Tool**: Perfect for learning chess strategy and tactics

## 🔧 Current Limitations

- Manual board scanning required
- Works with default 8x8 chess board sizing
- Console-based interface

## 📦 Requirements

- Python 3.7+
- Stockfish engine installed on your system
- Required Python packages (see `requirements.txt`)

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/ambarmandloi12-pixel/chess-with-stockfish-ban-free.git
cd chess-with-stockfish-ban-free
```

### 2. Install Stockfish

**Ubuntu/Debian:**
```bash
sudo apt-get install stockfish
```

**macOS (using Homebrew):**
```bash
brew install stockfish
```

**Windows:**
Download from [Stockfish Official](https://stockfishchess.org/download/)

### 3. Install Python Dependencies

```bash
pip install -r requirements.txt
```

## 📖 Usage

Run the main program:

```bash
python main.py
```

Follow the console GUI prompts to:
1. Input a chess position
2. Analyze the position with Stockfish
3. View engine recommendations
4. Study the analysis
